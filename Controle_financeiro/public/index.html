📖 Como Usar Este Sistema

  1. Servidor Backend: O sistema agora usa um banco de dados SQLite próprio rodando em Node.js
  2. Dados Persistentes: Todas as transações são salvas no arquivo financeiro.db
  3. Três Tipos de Transação:
    • Receita: Dinheiro que entra (salário, freelance, etc.) - aumenta o saldo
    • Despesa: Dinheiro que sai (contas, compras, etc.) - diminui o saldo
    • Investimento: Dinheiro aplicado (ações, fundos, etc.) - não afeta o saldo disponível
  4. Saldo Disponível: É calculado como Receitas - Despesas (investimentos são contabilizados separadamente)
  5. API REST: O frontend se comunica com o backend através de requisições HTTP
  6. Offline/Online: Veja o status da conexão no badge acima
💾 Salvamento Automático: Todos os seus dados são salvos automaticamente no banco de dados do servidor e permanecerão seguros!
